# SuperStore Sales Dashboard | Power BI Project

## Project Overview
Developed an interactive Power BI dashboard to analyze SuperStore sales data and generate actionable business insights. The dashboard provides a comprehensive view of sales performance, profit trends, customer segments, regional performance, shipping modes, and category-wise analysis through dynamic visualizations and filters.

## Objectives
- Monitor overall sales and profit performance.
- Analyze customer segments and purchasing behavior.
- Identify top-performing regions and categories.
- Track monthly sales trends and patterns.
- Enable data-driven decision-making through interactive dashboards.

## Tools & Technologies
- Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel
- Data Visualization
- Business Intelligence

## Dashboard Features
- KPI Cards for Sales, Profit, Orders, and Ship Days
- Sales Analysis by Region
- Customer Segment Analysis
- Payment Mode Analysis
- Category and Sub-Category Performance
- Monthly Sales Trend Analysis
- Interactive Slicers and Filters
- Geographical Sales Visualization

## Key Insights
- Consumer segment contributes the highest share of sales.
- COD is one of the most frequently used payment methods.
- Significant variations exist across regions and categories.
- Monthly trend analysis helps identify seasonal business patterns.

## Learning Outcomes
- Performed end-to-end data analysis using Power BI.
- Applied Power Query for data cleaning and transformation.
- Created calculated measures and business metrics using DAX.
- Developed interactive dashboards for business reporting.
- Implemented time-series analysis to identify sales trends.
- Enhanced data storytelling and visualization skills.
